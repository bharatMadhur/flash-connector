First API Call

After completing setup, submit jobs and poll by `job_id`.

  -H \"x-api-key: fc_xxx\" \\
  -H \"content-type: application/json\" \\
  -d '{\"input\":\"Hello\"}'\n\ncurl http://localhost:8000/v1/jobs/<job_id> \\
  -H \"x-api-key: fc_xxx\"
